---

## Prerequisites

- **Go 1.25+**
- **macOS or Linux**

## Install

```bash
git clone https://github.com/infisical/agent-vault.git
cd agent-vault
make build
```

This builds the frontend (React/Vite) and the Go binary. The output is a single `agent-vault` binary in the project root.

## Start the server

```bash
./agent-vault server
```

You'll be prompted to set a master password — this encrypts all credentials at rest using AES-256-GCM with an Argon2id-derived key.

To run in the background:

```bash
./agent-vault server -d
```

To stop a background server:

```bash
./agent-vault server stop
```

## Register the owner

The first user to register becomes the owner with full admin privileges.

```bash
./agent-vault register
./agent-vault login
```

## Verify

```bash
curl http://localhost:14321/health
```
